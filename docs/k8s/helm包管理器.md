## Helm安装

1. 安装包下载

   从github中下载helm安装包[Releases Page](https://github.com/helm/helm/releases)

2. 解压安装包

   ```shell
   [root@root ~/helm]# tar -zxvf helm-v3.5.0-linux-amd64.tar.gz
   ```

3. 将解压后的文件复制到`/usr/local/bin/helm`目录

   ```shell
   [root@root ~/helm]# tar -zxvf helm-v3.5.0-linux-amd64.tar.gz
   ```
